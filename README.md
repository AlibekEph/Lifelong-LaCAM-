# Lifelong Multi-Agent Pathfinding: Lifelong LaCAM Algorithm

## Project Goals

This project aims to:
1. Implement a naive adaptation of the LaCAM algorithm for the lifelong MAPF (Multi-Agent Path Finding) setting.
2. Evaluate the developed approach using existing lifelong MAPF benchmarks.
